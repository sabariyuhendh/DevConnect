## ğŸ› Troubleshooting

### Module Won't Resize
- Make sure you're dragging from edge/corner
- Check if you've reached minimum size
- Try refreshing the page

### Can't Create Room
- Check if room name already exists
- Ensure name doesn't have special characters
- Try a different name

### Timer Not Starting
- Check if another timer is running
- Try resetting the timer
- Refresh the page

### Module Disappeared
- Check if it's off-screen (drag it back)
- Click module icon in sidebar to reopen
- Refresh page to reset positions
